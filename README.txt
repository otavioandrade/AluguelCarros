O programa oferece ao usuário uma forma de encontrar a locadora de carros com o menor preço
de acordo com a quantidade de pessoas a utilizar o carro e o intervalo de dias.


O usuário entra com os seguintes dados para realizar a busca:
    Tipo de cliente (premium/normal);
    Quantidade de pessoas;
    Datas que ficará com o carro.
E receberá como resposta a locadora de carros e a lista de carros que essa locadora oferece.


Os dados devem ser inseridos no seguinte formato:

    <tipo de cliente>:<quantidade de pessoas>:<data,data,data...>


Exemplo:

entrada:    premium:2:10mar2009(ter),11mar2009(qua)

saída:      SouthCar:Uno
